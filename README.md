# Azure Resource Status Checker

A Bash script that audits Azure subscriptions by listing resource groups and their resources.

## Tech Used
- Bash
- Azure CLI

## Why This Matters
Demonstrates cloud automation, scripting, and CLI proficiency for DevOps roles.


## Preflight Checks
The script validates:
- Bash version
- Azure CLI availability
- Azure authentication status
- Active subscription context

This prevents runtime failures and ensures safe execution.
